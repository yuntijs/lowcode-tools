## ‚ú® Features

- [x] üöÄ Supports parsing components into material descriptions that conform to low code Engine Material Agreement specifications.
- [x] üöÄ Supports configuration to generate a single component material.
- [x] üöÄ Supports the configuration of filtering some components to generate materials.

## üì¶ Installation

To install @yuntijs/lowcode-materails-init, run the following command:

```bash
pnpm add @yuntijs/lowcode-materails-init
```

## üåê Usage

init.mjs

```mjs
import init from '@yuntijs/lowcode-materails-init';

init({
  components: undefined,
  skipComponents: ['TreeNode', 'DirectoryTree', 'Item', 'MonacoDiffEditor', 'BaseMonacoEditor'],
});
```

pakage.json

```json
{
  "scripts": {
    "init": "node init.mjs"
  }
}
```
